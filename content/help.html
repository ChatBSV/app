1. Commands

[ENTER] - Submit your prompt.
[SHIFT+ENTER] - Skip a line.

/MEME {Prompt} - Generate memes.
/IMAGINE {Prompt} - Generate images.
/IMAGINE {Prompt --format:} - Format images.

/GPT 3 - Set GPT-3.5 Turbo model.
/GPT 4 - Set GPT-4 model.
/DALL-E 2 - Set DALL-E 2 model.
/DALL-E 3 - Set DALL-E 3 model.

Default Models: GPT 3.5 & DALL-E 3.

/Help - Get information.

2. How It Works

Using Commands: To switch between models use the model name and number starting with /gpt or /dalle. /GPT 3 - Sets GPT-3.5 Turbo as the model. /GPT 4 - Sets GPT-4 as the model. >Use "/GPT 4" to switch to GPT 4 and continue the conversation. /DALLE 2 - Sets DALL-E 2 as the model. /DALLE 3 - Sets DALL-E 3 as the model. >Use "/DALLE 2" to set DALL-E 2 and then use /IMAGINE {Prompt} to generate an image with DALL-E 2. Generating Media Once you've set your preferred AI model, you can use commands like /IMAGINE and /MEME to generate content: /IMAGINE - Generate images using AI. >Type "/IMAGINE" followed by a prompt describing your image. /MEME - Create a unique AI generated meme. >Type "/MEME" followed by a prompt describing your meme. Image Formatting You can further customize your outputs with optional formatting parameters: /IMAGINE [prompt] --format 512 (Dimensions valid for DALL-E 2) /IMAGINE [prompt] --format horizontal (Horizontal and vertical formats are valid for DALL-E 3) >For example, type: "/imagine a futuristic city --format Vertical" Text and Code Formatting You can ask GPT to retun its response using Markdown for formatting. >For example: "Using markdown, write an article about blockchain scalability."
